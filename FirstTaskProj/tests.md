# Тестирование API
## Группа тестов №1: Тестирование GET
- Проверка на получение правильных данных (соответствующих DbSeedr) из БД 
- Проверка на возвращение статуса Ok после завершения операции
- Исполнение указанных выше проверок для всех видов GET (GET по модели, GET по модели и id, GET по FullCountryView)
- Проверка на корректность работы пагинации
## Группа тестов №2: Тестирование PUT
- Проверка на отправление и корректное обновление данных в БД
- Проверка на возвращение статуса Ok после отправления валидных данных в БД
- Проверка на возвращение статуса NotFound после отправления запроса на изменение несуществующих элементов в БД
## Группа тестов №3: Тестирование POST
- Проверка на отправление и корректное добавление данных в БД
- Проверка на возвращение статуса Ok после отправления валидных данных в БД
- Проверка на возвращение статуса BadRequest после отправления null объекта
## Группа тестов №4: Тестирование DELETE
- Проверка на корректное удаление данных из БД
- Проверка на возвращение статуса Ok после отправления валидных данных в БД
- Проверка на возвращение статуса NotFound после отправления запроса на удаление несуществующих элементов в БД
