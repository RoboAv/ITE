# Спроектировать и разработать REST-api для системы: страны-регионы-города
## Основные пункты
- Схема БД (использовать Microsoft SQL-сервер)
- Реализация схемы в EF code-first (узнать database-first)
- Простая view в SQL через миграцию (и зачем)
- Модель представления для view
- API для чтения с paging-ом
- Вяснить разницу между СУБД: MS SQL, PostgreSQL, MySQL
*Можно подключить свагер для UI для RESTа*
